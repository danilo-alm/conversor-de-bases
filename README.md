# Conversos de Bases
Converte um número de qualquer base x para outra base y.</br>
## Exemplo
```
Base de Origem: 10
Número: 10.234
Base de Destino: 16
A.3BE76C8B439582
```
Erros de precisão podem acontecer com o uso de números com casas decimais devido à natureza imprecisa dos floats na programação
